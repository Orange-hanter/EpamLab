Создать контейнерный  целочисленный класс-массив (или любой собственный класс-например, кинофильмов), реализуя функционал контейнеров в языке С++
1. Должен отслеживать два значения: данные и свою длину, использовать указатель для хранения данных.
2. Конструктор для создания пустого массива.
3. Конструктор для создания массива заданного размера.
4. Функция для очистки массива.
5. Деструктор.
6. Должен иметь доступ к элементам массива (перегрузка []). Контроль корректности индекса.
7. Функция доступа для возврата длины массива.
8. Функция для изменения размера массива.
9. Функция для добавления/удаления элемента.
